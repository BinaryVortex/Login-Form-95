# Login Form 95

Login Form built using plain HTML, CSS and a bit of JavaScript.

![Login Form Screenshot](https://raw.githubusercontent.com/BinaryVortex/Login-Form-95/main/Screenshot%202024-08-06%20085904.png)

## Overview

A clean, responsive login form implemented with semantic HTML and modern CSS. This repository is a small, framework-free example that you can use as a starting point for personal projects, demos, or learning front-end layout and styling.

## Features

- Simple, attractive login UI
- Responsive layout for desktop and mobile
- Lightweight — no frameworks required
- Easy to customize: colors, fonts, and layout are controlled in `style.css`

## Files

- `index.html` — main page containing the login form markup
- `style.css` — styling for the form and page
- `Screenshot 2024-08-06 085904.png` — demo screenshot used in this README
- `fantastic-mosque-architecture-islamic-new-year-celebration (3).jpg` — extra image asset in the repo

## Preview / Usage

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-95.git

2. Open `index.html` in your browser:
   - Double-click `index.html`, or
   - Serve locally (recommended for some browsers): `npx http-server` or `python -m http.server` then open http://localhost:8080

## Customization

- Colors and layout: edit `style.css`
- Form markup and fields: edit `index.html`
- Replace images by updating the files in the repository and adjusting paths in the HTML/CSS

## Accessibility & Responsiveness

The form uses semantic HTML and responsive CSS rules to adapt across screen sizes. Suggestions to improve accessibility:

- Add explicit `label` elements tied to inputs with `for` attributes when necessary
- Add ARIA attributes where appropriate (e.g., `aria-invalid`) for enhanced screen-reader support
- Ensure visible focus outlines for keyboard navigation
